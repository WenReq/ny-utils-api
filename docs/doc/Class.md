# Class

入参：

- element：需要操作的 DOM 元素
- class：需要操作的 Class 名

## addClass

> 为元素添加 class

```js
import { addClass } from "ny-utils";

addClass(element, class);
```

## hasClass

> 判断元素是否有某个 class

```js
import { hasClass } from "ny-utils";

hasClass(element, class);
```

## removeClass

> 为元素移除 class

```js
import { removeClass } from "ny-utils";

removeClass(element, class);
```
